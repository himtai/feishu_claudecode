# 飞书 Claude Agent 项目总结

## 项目概述
这是一个将 Claude AI 集成到飞书（Feishu）的智能助手项目，让用户可以在飞书中直接使用 Claude 的完整编程能力。

## 核心功能

### 1. 完整的 Claude Code 能力
- 文件读取、编辑、创建
- 执行 Shell 命令
- 代码搜索和分析
- 网络搜索和抓取

### 2. 实时反馈系统
- 工具调用过程可视化
- 执行状态实时更新
- 错误信息即时提示

### 3. 会话管理
- 支持多轮对话
- 上下文自动保持
- 可随时重置会话

## 快速启动

1. 安装依赖：`pip install -r requirements.txt`
2. 配置环境：复制 `.env.example` 为 `.env`
3. 运行服务：`python feishu_agent_bot.py` 或 `./start.sh`
4. 配置飞书事件订阅

## 特殊命令

- `/help` - 显示帮助信息
- `/cancel` - 取消当前任务
- `/reset` - 重置会话
- `/status` - 查看状态

## 典型使用场景

### 场景 1：Bug 修复
用户描述问题 → Agent 读取代码 → 搜索日志 → 修复代码 → 运行测试

### 场景 2：添加功能
用户提需求 → Agent 分析结构 → 创建新文件 → 更新配置 → 编写测试

### 场景 3：代码审查
用户指定 PR → Agent 获取差异 → 分析代码 → 查找最佳实践 → 生成报告

## 权限管理

Agent 在执行危险操作时会请求确认：
- 删除文件
- 修改关键配置
- 执行系统命令
- 网络请求

## 可用工具集

- **文件操作**: Read, Write, Edit
- **命令执行**: Bash
- **搜索**: Glob, Grep
- **网络**: WebSearch, WebFetch
- **子代理**: Agent

## 部署方式

1. **Docker 部署** - 容器化运行
2. **Systemd 服务** - Linux 系统服务
3. **云服务部署** - 阿里云/腾讯云/AWS

## 安全建议

1. **API Key 安全** - 使用环境变量，定期轮换
2. **文件访问控制** - 限制 PROJECT_ROOT 范围
3. **命令执行限制** - 审查所有 Bash 命令
4. **网络安全** - 使用 HTTPS，配置白名单

## 常见问题

- **多用户支持**: 是，每个用户独立会话
- **会话保持**: 直到用户重置或服务重启
- **语言支持**: 所有编程语言
- **私有仓库**: 支持（需配置密钥）

## 技术特点

- 基于 Claude Agent SDK
- WebSocket 实时通信
- 异步消息处理
- 完整的工具调用可视化
- 灵活的权限控制
